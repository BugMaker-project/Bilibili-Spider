在程序start目录下，你将可以看到5个文件:
--consoleVersion.cmd:打开命令行模式下的该工具。
--downloadTools.cmd:下载视频。如果失败，请修改src目录下的setting.xml
--init.cmd:第一次运行前请执行，安装依赖库.
--windowsVersion.cmd:打开qt界面版本的该工具.(加载较慢)
--教程.cmd：一点小提示，没啥用途.

使用cd命令在命令行下进入src目录. 

cd ..\src

使用命令行方式(以在本目录下为例.): 
python enterPoint.py --console

使用qt界面的打开方式： 
python enterPoint.py --windows

下载工具: 
python enterPoint.py --download BVxxxx
标签 默认内容 更改内容
setupPackage-mirror https://mirrors.aliyun.com/pypi/simple/ 任何你喜欢的pypi镜像
parameter-avbv bvid 可更换为aid或bvid.不同的链接格式
parameter-favicon favicon.ico 可以更换你任何想用的图标。不过后缀名必须是.ico
videoFormat flv 可以修改你想要的视频格式，限定为flv,flv720,flv480,flv360
